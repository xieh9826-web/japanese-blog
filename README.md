# 🧪 AISlime — AI 前沿动态聚合

每日精选全球 AI 行业最新动态，支持中文/日本語/English 切换。

## 功能

- 🌐 **多语言** — 中/日/英 即时切换
- 🌓 **深色/浅色主题** — 跟随系统或手动切换
- 🔥 **热度追踪** — 基于多维度热度评分
- 📱 **响应式** — 桌面 & 移动端适配
- 🔍 **搜索过滤** — 按分类和关键词筛选

## 技术栈

- HTML5 / CSS3 (CSS Variables + Flexbox/Grid)
- Vanilla JavaScript (i18n, Theme, Filter)
- GitHub Pages

## 本地运行

```bash
cd ~/japanese-blog
python3 -m http.server 8000
# → http://localhost:8000
```
